# Mutation Signature Explorer

Hosted: [https://imuse.lrgr.io](https://imuse.lrgr.io)

### Dependencies
- nodejs
- yarn (`npm install -g yarn`)
- `yarn install`

### Run
[http://localhost:8080](http://localhost:8080)
```
yarn serve
```

### Build
```
yarn build
```
